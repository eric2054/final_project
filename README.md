## 遊戲玩法
## 1.先由擁有梅花7的玩家先出牌。
## 2.下一位玩家必須出梅花6或梅花8,也能夠出其他花色的7。
## 3.如果以上的都沒有則選一張蓋牌
## 4.最後計算玩家蓋牌的點數,點數越小的贏,沒有蓋牌的就是第一名
